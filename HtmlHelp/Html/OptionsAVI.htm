AVI Options

Use AVI archive
Do you want all captured images to be stored in a uncompressed AVI file?

Filename
The filename for the AVI archive. It can have the same tags as the text captions, so the filename can be date and time related. Remember that a '/' and ':' are invalid characters in a filename, therefore %G and %g are available. The time stamp is constructed when the file is first created.

Frames per second
The number of frames per second in the AVI. This is not related to the frames per seconds of the preview window.

New file at midnight
Do you want a new file AVI file at midnight. If you do use this options, it is advised to use time related tags in the filename.

Know that only pictures taken by an enabled Store Event are stored in the archive. Pictures taken by the web server are not stored.
