FTP Storage

Name
The unique name of this storage event. You can only change the name when you create a new item.

Enable
Is this event enabled?

Interval X seconds/minutes/hours
The time interval between between captures.

Host
This is the internet address of your FTP server.

Port
This is IP port to connect to on your FTP server.

Username
The username to use when logging on to the FTP server.

Password
The password to use when logging on to the FTP server. The password is stored in the registry, with a relatively simple encryption mechanism. There is a potential security risk. If someone has access to your registry and they could break the encryption, this will give them access to your ftp account.

File
This is the filename that you wish to store on the server. You may need to include a directory in the filename. It can have the same tags as the text captions, so the filename can be date and time related. Remember that a '/' and ':' are invalid characters in a filename, therefore %G and %g are available. The time stamp is constructed when the file is uploaded to the server.

Keep connection open between stores
Set this option to keep the FTP connection opened between the stores.

Use RAS
Do you want to check for an active RAS connection (Remote Access Service, also known as Dial-up connections) before the upload is attempted.

Passive mode
Use FTP in active or passive mode.

Connection
Specify the RAS connection to look for.

Reset time when not stored
Reset the timer when there has been no storage. This is most likely to happen when there has been no motion detected or there has been no RAS connection.

Custom captions
When you want to use custom captions for this store event, you need to check this option and create or adjust the custom captions with the Settings button.
