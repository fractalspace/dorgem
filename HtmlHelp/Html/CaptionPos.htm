Caption Position

Relative
Use for easy positioning of the caption. Till version 2 this was the only way to handle positioning of the caption.

Absolute
Specify the position of this caption in absolute pixels. The co-ordinate 0,0 is top-left.

Horizontal
The horizontal position

Vertical
The vertical position

When you press the OK button, the position is saved directly in the caption, even when Cancel is pressed in the caption settings itself.
