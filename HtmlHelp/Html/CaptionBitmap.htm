Bitmap Caption

Name
The unique name of this caption. You can only change the name when you create a new item.

Enable
Is this caption enabled?

Bitmap
The file that contains the bitmap.

Type
The type of overlay

  • Transparent
    For transparent overlay, you can select the color that has to be made transparent.

  • Multiply

  • Copy
    The bitmap is placed fully over the captured bitmap.

Transparent
Select the color to be used as transparent

Position
Specify the position of the bitmap.
