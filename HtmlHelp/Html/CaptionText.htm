Text Caption

Name
The unique name of this caption. You can only change the name when you create a new item.

Enable
Is this caption enabled?

Length
Specify the maximum number of characters of the caption (0=no limit).

Text
Give the text to put on the image. This text can be time or date related by using these tags.

Language
Select the languages the time/date related tags must be in.

File
Read the first line of this text file. So you can generate a text file from another application with dynamic information (like weather info or the Winamp now playing plug-in) and put it on the image.

Tranparent
If this is checked then the text caption is put on the image transparently, otherwise the text is drawn in a box.

Shadow
When the text is drawn transparently, Full will put a one pixel shadow around the text, Drop will put a single pixel shadow under the text.

Position
Specify the position of the bitmap.

Font
Select the font of the text.

Back Color
Select the background color of the text.

Fore Color
Select the foreground color of the text.

You can specify your own text or a read it from a file. Therefore only as soon as in one of these fields a character is entered the other one will be disabled.
